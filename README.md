## What I Built
- Generated 3.5M synthetic transaction records using Python
- Designed star schema with 3 tables (Transactions, Customers, Merchants)
- Created 8 interactive visuals in Power BI
- Built KPIs, trend charts, and payment distribution analysis
- Added date and payment method slicers for filtering

## Key Insights
- UPI dominates at 45% of all transactions
- Top 10 merchants drive 30% of revenue
- 88% overall success rate across payment methods
- Clear seasonal patterns in Q4 2024
